# itiGit2
## How to remove my dev branch from remote server :
   `git push origin :dev`
   
## How to remove my dev branch locally :
   `git branch -D dev`


## How to list tags locally :
   `git tag`
   
## How to remove tag locally :
   `git tag -d tagName`

## How to remove tag from remote server :
   `git push origin --delete tagName`
